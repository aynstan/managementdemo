## 功能特点：  

1.可插拔的权限系统，权限控制粒度可精确页面和按钮，任何页面不需要添加一句权限处理代码，只需继承基类即可。

2.非常好的安全性

3.良好的可维护性，可扩展性

4.适用范围：一般的中小型系统

5.项目架构：普通三层架构
